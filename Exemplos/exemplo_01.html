A disciplina de Introdução à Construção de Páginas Web

A disciplina é muito boa. Aprenda a usar HTML e CSS para criar páginas web. HTML é a linguagem de marcação com a qual você envolve o conteúdo, para instruir os navegadores sobre cabeçalhos, listas, tabelas, etc. CSS é a linguagem de estilos que você usa para estilizar a página, para instruir os navegadores a alterar a cor, a fonte, o layout e muito mais.
